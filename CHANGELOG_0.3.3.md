# FALSE ACCESS 0.3.3 — MOBILE / PWA

## Settings

- отдельное приложение Settings;
- ручное сохранение на устройство;
- экспорт сохранения в JSON;
- импорт сохранения из JSON;
- подтверждение перед полным сбросом;
- принудительная проверка обновления;
- статус сети, PWA и локального хранилища.

## Offline

- удалена зависимость от Google Fonts;
- новый service worker кэширует production-сборку, сайты мира и Pyodide;
- навигация работает через offline fallback;
- старые кэши очищаются при обновлении версии.

## Mobile

- Terminal получил переключатель «Максим / Терминал»;
- Code Editor получил переключатель «Максим / Код»;
- обучение больше не скрывается мобильными стилями;
- улучшены размеры редактора, терминала, кнопок и safe-area;
- Settings полностью адаптированы под телефон.

## Identity

- новый SVG-логотип, созданный кодом;
- отдельная maskable-иконка для PWA;
- логотип добавлен на загрузочный экран, мобильный рабочий стол и Settings;
- добавлены PWA и Apple mobile meta-теги.
