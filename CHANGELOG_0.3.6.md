# FALSE ACCESS 0.3.6 — Mission Runtime

- базовые приложения отделены от сюжетных слоёв;
- CLINIC-01 перенесена в единый Mission Runtime;
- добавлена вводная сцена с Максимом и контекст клиники;
- Messenger на телефоне получил список диалогов, отдельный разговор и звонки по этапам;
- Mail на телефоне разделена на список и открытое письмо;
- Terminal, Code и SIEM очищаются от сюжетных панелей после прохождения этапа;
- после каждого этапа появляется переход к следующему приложению;
- Missions показывает только текущую цель и текущий диалог;
- отчёт собирается из правильных и неправильных вариантов;
- мобильный Code прокручивается целиком;
- поля на телефоне используют 16px и не вызывают автоматическое приближение iOS;
- сохранения мигрированы на progress-v4.
