# FALSE ACCESS 0.3.4

## Мобильный интерфейс

- исправлена пустая вкладка разговора в Terminal;
- текущая реплика и команда теперь видны над shell до ввода;
- текущая строка Python и её назначение видны над редактором до вставки;
- редактор кода нормально прокручивается пальцем;
- OUTPUT свёрнут по умолчанию и раскрывается только при наличии результата;
- главное меню плотнее заполняет экран и не оставляет большой пустой блок снизу.

## Название и PWA

- из интерфейса убран графический логотип;
- `FA//OS` и `FA//MOBILE` заменены на `FALSE ACCESS`;
- имя установленного веб-приложения — `FALSE ACCESS`;
- добавлены отдельные PWA-иконки 192, 512, maskable и Apple Touch Icon;
- новая иконка построена SVG-кодом: окно редактора, скобки и строки кода;
- офлайн-кэш обновлён до версии 0.3.4.
