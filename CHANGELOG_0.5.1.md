# FALSE ACCESS 0.5.1 — Progression Framework

## Добавлено

- единая прогрессия от этапа 0 до этапа 8;
- отдельные значения теории, практики с подсказками, самостоятельной практики и рабочего опыта;
- 20 направлений подготовки;
- приложение Career;
- требования для следующего этапа;
- список пробелов игрока;
- итоговые проверки для каждого этапа;
- требования этапа и конкретной практики для вакансий;
- требования этапа для контрактов разной сложности;
- выбор двух специализаций после этапа 3;
- новые вакансии для поздних этапов;
- миграция старых сохранений на schemaVersion 2.

## Рабочий стол

- приложения разделены на постоянные и временные;
- Interview появляется только после CLINIC-01 и исчезает после собеседования;
- First Shift появляется после принятия работы и исчезает после первой смены;
- открытое временное приложение автоматически закрывается после завершения этапа;
- временные приложения отмечены как STORY.

## Сохранения

- новый ключ сохранения: `false-access-progress-v6`;
- сохранения 0.5.0 и более старых версий импортируются автоматически;
- сюжет, деньги, жильё, работа, контракты и навыки сохраняются.
