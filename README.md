# FALSE ACCESS — MVP 0.3.6

Сюжетная RPG и браузерный тренажёр по информационной безопасности.

## Что работает

- виртуальная ОС для ПК и телефона;
- Mission Runtime: приложения существуют отдельно, миссия временно добавляет цель, диалог и проверку;
- CLINIC-01 с вводным звонком, Terminal, Python, SIEM, отчётом и завершением разговора;
- техническое собеседование, предложение о работе и первая смена;
- Messenger и Mail с отдельной мобильной навигацией;
- повторяемые контракты по Linux, Python, сетям, SOC и Web Security;
- локальное сохранение, импорт/экспорт, PWA и офлайн-кэш.

## Запуск

```bash
npm ci --no-audit --no-fund
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
