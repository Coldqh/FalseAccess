# FALSE ACCESS

Браузерная сюжетная RPG и симулятор карьеры в информационной безопасности.

## 0.5.0

В игре появился постоянный симулятор жизни: время, состояние, питание, жильё, техника, работа, деньги, репутация и розыск. Сюжетные миссии и повторяемые лаборатории работают поверх этой системы.

## Запуск

```bash
npm ci --no-audit --no-fund
npm run dev
```

## Сборка

```bash
npm run build
```

Проект публикуется через GitHub Pages и работает офлайн после первой полной загрузки.
