# FALSE ACCESS — 0.4.1

Браузерная сюжетная RPG и локальный тренажёр по информационной безопасности.

## Что есть

- пролог CLINIC-01;
- Linux-терминал и виртуальная файловая система;
- Python через Pyodide;
- SIEM и конструктор отчёта;
- техническое собеседование;
- первая смена в SOC;
- фишинг, Windows Event 4688, PowerShell и DNS;
- Mail, Messenger, Work Queue и сайты мира;
- локальные сохранения и офлайн-PWA.

## Запуск

```bash
npm ci
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
