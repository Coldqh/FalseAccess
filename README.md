# FALSE ACCESS

Браузерная сюжетная RPG и симулятор карьеры в информационной безопасности.

## 0.5.1

Игра получила общую прогрессию 0–8, отдельный учёт теории и практики, экран Career, требования вакансий и контрактов, итоговые проверки и специализации. Сюжетные приложения теперь временные и исчезают после завершения своего этапа.

## Запуск

```bash
npm ci --no-audit --no-fund
npm run dev
```

## Сборка

```bash
npm run build
```

Проект публикуется через GitHub Pages и работает офлайн после первой полной загрузки.
