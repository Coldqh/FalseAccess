# FALSE ACCESS — MVP 0.3.5

Сюжетная браузерная RPG и практический тренажёр по информационной безопасности. Версия 0.3 рассчитана на игрока без знаний Linux, Python и ИБ.

## Порядок прохождения

1. Пройти `Academy Zero`.
2. Открыть `Missions` и начать `CLINIC-01`.
3. Выполнить 6 шагов в `Terminal`.
4. Собрать первую Python-программу из 8 объяснённых частей.
5. Разобрать алерт в `SIEM`.
6. Сдать технический отчёт в `Notes`.
7. Пройти `Technical Interview`.
8. Принять предложение в `Mail`.
9. Закрыть первый фишинговый тикет в `First Shift`.
10. Выполнять повторяемые заказы через `Work Queue`.

## Что работает

- виртуальная браузерная ОС;
- ПК и мобильный интерфейс;
- локальные сохранения с миграцией v1/v2 → v3;
- 7 базовых уроков Academy;
- обучающий shell на xterm.js;
- реальные по синтаксису Linux-команды внутри безопасной виртуальной системы;
- Python через Pyodide/WebAssembly;
- SIEM triage;
- технический отчёт;
- техническое собеседование;
- первый рабочий день и фишинговый тикет;
- почта, мессенджер, сайты мира;
- процедурные заказы по Linux, Networking, Python, SOC и Web Security;
- деньги, репутация и история заказов.

## Локальный запуск

```bash
npm ci --no-audit --no-fund
npm run dev
```

## Production-сборка

```bash
npm run build
npm run preview
```

## GitHub Pages

Workflow `.github/workflows/deploy.yml`:

- использует Node.js 22.16.0;
- проверяет отсутствие внутреннего npm registry в lock-файле;
- устанавливает зависимости через публичный `registry.npmjs.org`;
- выполняет `npm ci --no-audit --no-fund`;
- собирает `dist`;
- публикует Pages.

Все цели, IP-адреса, домены, письма и данные вымышлены. Игра не обращается к внешним системам.
