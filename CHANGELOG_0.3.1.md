# FALSE ACCESS 0.3.1

## Обучение встроено в миссию

- Academy Zero убрана с основного рабочего стола и больше не является обязательным этапом.
- первая глава начинается с защищённого звонка Максиму Белову;
- теория выдаётся короткими репликами только перед нужным действием;
- после каждой команды Максим реагирует на фактический результат;
- Terminal совмещает разговор, текущее действие и рабочий shell;
- Python-лаборатория оформлена как демонстрация экрана с куратором;
- каждая строка программы сразу решает часть задачи CLINIC-01;
- самостоятельный режим сохранён для опытных игроков;
- обновлены Missions, Mail, Skills, onboarding и справка терминала.
