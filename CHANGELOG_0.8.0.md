# FALSE ACCESS 0.8.0 — DEADFRAME-08

## Новая техническая глава

- форензика дискового образа E01;
- SHA-256 и сохранение целостности доказательств;
- таблица разделов, файловая система и удалённые записи;
- bodyfile и файловая временная линия;
- browser downloads, Prefetch, registry Run keys и LNK;
- восстановление удалённого журнала через forensic clone;
- анализ RAM через Volatility 3;
- процессы, дерево процессов, command line, handles и netscan;
- связывание дисковых и memory-артефактов;
- chain of custody и безопасная локализация;
- второй образ без готовой последовательности;
- итоговый DFIR-отчёт и скрытый экзамен `disk-memory-forensics`;
- повторяемые контракты по timeline и удалённым артефактам.

## Мобильные исправления

- автоматическая проверка обновлений больше не блокирует интерфейс;
- фоновой проверке задан жёсткий таймаут;
- тяжёлый Pyodide больше не скачивается во время установки Service Worker;
- уведомление обновления не перекрывает мобильные кнопки;
- баланс отображается в мобильной строке состояния;
- открытое письмо в Mail больше не приводит к пустому чёрному экрану;
- reader Mail нормально прокручивается и имеет постоянную кнопку возврата.

## Техническое

- версия приложения и PWA-кэша: `0.8.0`;
- сохранения `v16` автоматически мигрируют на `v17`;
- временное приложение DEADFRAME-08 исчезает после завершения.
