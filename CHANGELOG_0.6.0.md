# FALSE ACCESS 0.6.0 — MARSHRUT-01

Первая полноценная свободная сюжетная операция после первой смены.

## Геймплей

- встреча с Игорем в кафе «Сигнал»;
- временное приложение операции, которое исчезает после завершения;
- настоящий игровой сайт службы доставки «Маршрут»;
- разбор HTTP-методов, кодов ответа, cookies и sessions;
- терминальная работа с access.jsonl, sessions.jsonl и audit.jsonl;
- реальный Python через Pyodide;
- объединение нескольких журналов в временную линию;
- выбор доказанных фактов;
- конструктор технического отчёта;
- шесть финальных решений с разной оплатой, репутацией и риском.

## Симулятор

- операция тратит два периода игрового времени;
- результат влияет на деньги, Игоря, «Сферу», цифровой след и криминальную известность;
- после главы открываются повторяемые контракты по анализу сессий;
- прогресс Web, Python, Networking, SOC и Communication начисляется отдельно.

## Сохранения

- новое хранилище false-access-progress-v8;
- автоматическая миграция сохранений 0.5.3 и старше;
- этапы операции, код, ответы и терминальные задачи сохраняются.
