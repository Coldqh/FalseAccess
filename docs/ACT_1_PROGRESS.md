# FALSE ACCESS — Акт 1 «Сфера»

## Готово

- обязательный маршрут после Акта 0;
- слой 1: сюжетная учебная SOC-миссия с mail, endpoint, DNS, identity и CMDB;
- alert queue с переходом к исходным артефактам;
- semantic assessment по действиям, hypotheses, evidence, decision и handoff;
- два допустимых containment-пути;
- слой 2: три самостоятельных seeded-контракта без подсказок;
- слой 3: первая смена с четырьмя alert, одним легитимным сигналом и неизвестным маршрутом;
- переход в основную игру только после mastery check;
- совместимость со старым Act 2 через progress flags;
- локальная read-only среда, external network disabled, reset и reroll.
