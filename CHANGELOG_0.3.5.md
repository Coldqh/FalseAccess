# FALSE ACCESS 0.3.5

## Mobile Code Editor

- вся вкладка `Код` теперь прокручивается целиком;
- теория текущей строки находится прямо над редактором;
- добавлены кнопки `Вставить строку` и `Проверить готово`;
- вывод больше не перекрывает редактор и остаётся сворачиваемым.

## Terminal

- смысл команды и указание Максима показаны над терминалом до ввода;
- команда вставляется одним нажатием;
- после завершения терминала следующий этап явно ведёт в Code Editor.

## Звонок по CLINIC-01

- карточка вызова меняется по прогрессу дела;
- этапы: Terminal → Code → SIEM → Notes → Interview;
- кнопка больше не возвращает игрока в уже завершённый терминал.

## Контракты

- в каждый контракт добавлены готовые команды для вставки;
- Python-контракт получил вставку нужного блока кода;
- кнопка `Проверить готово` запускает проверку прямо из блока кода.

## Отчёт

- после разбора SIEM автоматически собирается черновик отчёта;
- его можно восстановить кнопкой `Заполнить по найденным фактам`;
- игроку остаётся проверить текст и отправить его.
