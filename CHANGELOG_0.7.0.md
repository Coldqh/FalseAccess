# FALSE ACCESS 0.7.0 — IRONROOT-06

## Новая глава

- полноценный корпоративный домен `IRONROOT.LOCAL`;
- пять связанных узлов: контроллер домена, jump-host, файловый сервер, приложение и рабочая станция;
- Active Directory: пользователи, компьютеры, группы, OU и делегированные права;
- Kerberos: TGT, service ticket, события `4768` и `4769`;
- Windows logon events `4624` и специальные права `4672`;
- SMB-сессии и восстановление движения между машинами;
- GPO, SYSVOL, startup-скрипты и ACL;
- сервисные учётные записи, SPN и опасная делегация;
- исправление PowerShell-сценария локализации;
- самостоятельная проверка второго доменного сегмента;
- итоговый отчёт и скрытый экзамен `stage-3-capstone`.

## Прогресс и контракты

- Active Directory добавлен в экран Skills;
- открываются процедурные задания по старым учёткам, группам, GPO и SYSVOL ACL;
- глава прокачивает AD, Windows, PowerShell, SIEM, IR, DFIR, Networking и Security Engineering;
- временное приложение `IRONROOT-06` исчезает после завершения;
- схема сохранений обновлена до `false-access-progress-v14`.

## Техническое

- версия приложения и PWA-кэша: `0.7.0`;
- сохранения `0.6.5` мигрируют автоматически;
- сборка проверена через чистый `npm ci` и `npm run build`.
